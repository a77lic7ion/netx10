Nexus AI

An intelligent desktop application for multi-vendor network automation and management.

Nexus AI is a sophisticated Windows application built for network professionals managing complex, multi-vendor environments. It combines a serial-based CLI terminal with a powerful AI assistant to revolutionize network automation, configuration, and troubleshooting.

‚ú® Core Features

  • Unified Multi-Vendor Management Seamlessly control devices from Cisco, H3C, Juniper, and Huawei through a single interface.
  • Intelligent CLI Terminal A feature-rich serial terminal with syntax highlighting, autocompletion, and command history.
  • AI-Powered Assistant Leverage an integrated AI to generate commands, draft configurations, and diagnose network issues.
  • Cross-Vendor Command Translation Instantly translate CLI commands between different vendor syntaxes (e.g., Cisco IOS to Juniper JunOS ).
  • Advanced Session Management Effortlessly manage and switch between multiple concurrent device sessions.
  • Configuration Templates Deploy standardized configurations quickly with a library of vendor-specific templates.

üöÄ Getting Started

Prerequisites

Ensure you have Python 3.9+ and Pip installed on your system.

Installation

1. Clone the repository:

git clone https://github.com/YourUsername/Nexus-AI.git
cd Nexus-AI

2. Install dependencies:

pip install -r requirements.txt

3. Run the application:

python src/main.py
