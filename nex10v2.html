Nexus AI

An intelligent desktop application for multi-vendor network automation and management.

Overview

Nexus AI is a sophisticated Windows application built for network professionals managing complex, multi-vendor environments. It combines a serial-based CLI terminal with a powerful AI assistant to revolutionize network automation, configuration, and troubleshooting.

Live CLI, AI Chat, and Session Management in a unified interface.

‚ú® Core Features

Unified Multi-Vendor Management

Seamlessly control devices from Cisco, H3C, Juniper, and Huawei through a single interface.

Intelligent CLI Terminal

A feature-rich serial terminal with syntax highlighting, autocompletion, and command history.

AI-Powered Assistant

Leverage an integrated AI to generate commands, draft configurations, and diagnose network issues in real-time.

Cross-Vendor Command Translation

Instantly translate CLI commands between different vendor syntaxes (e.g., Cisco IOS to Juniper JunOS).

Advanced Session Management

Effortlessly manage and switch between multiple concurrent device sessions.

Configuration Templates

Deploy standardized configurations quickly with a library of vendor-specific templates.

üöÄ Getting Started

Prerequisites

  • Python 3.9 or newer
  • Pip package installer

Installation

  1. 1. Clone the repository:

    git clone https://github.com/YourUsername/Nexus-AI.git
cd Nexus-AI

  2. 2. Install dependencies:

    pip install -r requirements.txt

  3. 3. Run the application:

    python src/main.py
